# Security Policy

## Supported versions

This is an educational project. There are no formal release channels or guaranteed support timelines.

## Reporting a vulnerability

If you believe you have found a security issue in this repository:

- Please open an issue with a clear description and reproduction steps.
- If you prefer private disclosure, contact the maintainer before publishing details.

## Safe usage reminder

This repository includes intentionally insecure demos (MD5 rainbow tables). They are included only to show why modern password hashing is required.

Do not use these demo techniques on systems or data you do not own or have explicit permission to test.
